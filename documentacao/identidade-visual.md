# FitPro360 - Identidade Visual

## Cores

### Primárias
- **Verde Energia**: `#00D4AA` — Ação principal, progresso, saúde
- **Dark Background**: `#0A0A1A` — Fundo principal
- **Accent Laranja**: `#FF6B35` — Destaque, calorias, ação secundária

### Secundárias
- `#1A1A2E` — Card background
- `#16213E` — Card alternativo
- `#0F3460` — Gradientes profundos
- `#E94560` — Alertas, frequência cardíaca
- `#FDCB6E` — Nutrição, avisos
- `#74B9FF` — Água, informação
- `#A29BFE` — Profissionais, premium
- `#55EFC4` — Sucesso, natureza

## Tipografia
- **Headings**: System font (bold 700)
- **Body**: System font (regular 400)
- **Escala**: 10, 12, 14, 16, 20, 24, 32, 40px

## Estilo Visual
- **Dark Mode First**: Background escuro com elementos glassmorphism
- **Glassmorphism**: Transparência + blur + borda sutil
- **Gradientes**: Linear gradients nos headers e CTAs
- **Micro-animações**: Transições suaves em cards e botões
- **Ícones**: @expo/vector-icons (Ionicons + MaterialCommunityIcons)

## Componentes Padrão
- Cards com `rgba(255,255,255,0.08)` + borda `rgba(255,255,255,0.15)`
- Botões primários com gradiente `#00D4AA → #00B894`
- Chips selecionáveis com borda colorida
- Badges com fundo transparente + cor

## Tom de Voz
- **Motivacional** mas não forçado
- **Direto** e sem rodeios
- **Inclusivo**: para todos os níveis
- Emojis em títulos de seção
